# 概要
この資料は、ユーザーのロールについての概要を示すものである。

# ロールの種類
本システムにおける、ユーザーのロールは3種である。

- 1.SuperAdmin
- 2.事務局
- 3.申出者

各ロールについて、以下に示す。

## SuperAdmin
- role_id は 1。
- 開発初期段階で設けた特権ロール。
- システムの全ての機能を利用出来る権限として設けた。
- 実際には利用されておらず、一部、実装もされていない。
- 不要であるため、削除することが望ましい。

## 事務局
- role_id は2。
- 申出の申請を受け付ける側のロール。
- 一般的な意味での、管理者アカウントの位置づけ。
- 国立がんセンターの職員に対して割り当てられるアカウントは、事務局アカウントである。

## 申出者
- role_id は3。
- 申出の申請を行う、システム利用者側のロール。
- 一般的な意味での、ユーザーアカウントの位置づけ。
- 申出者ロールは、自分が作成した申請の情報しか参照出来ない。
